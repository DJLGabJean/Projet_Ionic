A propos

Objectif de l'application

Minevolve est une application qui vous permet d'apprendre à créer des objets dans le jeu Minecraft. Vous pourrez apprendre à créer des objets en suivant des recettes, mais aussi en découvrant les propriétés des différents ingrédients.

Technologie utilisées

Cette appplication utilise le framework Ionic pour le front-end et le back-end. Elle utilise aussi une base de données pour stocker les recettes et les ingrédients sur une base de données Firebase.
